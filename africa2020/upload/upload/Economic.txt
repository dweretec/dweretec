<p>An <b>economy</b> consists of the <a href="http://en.wikipedia.org/wiki/Economic_system" title="Economic system">economic system</a> of a country or <a href="http://en.wikipedia.org/wiki/Other" title="Other">other</a> area, the <a href="http://en.wikipedia.org/wiki/Wage_labour" title="Wage labour">labor</a>, <a href="http://en.wikipedia.org/wiki/Capital_(economics)" title="Capital (economics)">capital</a> and <a href="http://en.wikipedia.org/wiki/Land_(economics)" title="Land (economics)">land</a> <a href="http://en.wikipedia.org/wiki/Resources" title="Resources" class="mw-redirect">resources</a>, and the <a href="http://en.wikipedia.org/wiki/Economic_agent" title="Economic agent" class="mw-redirect">economic agents</a> that socially participate in the <a href="http://en.wikipedia.org/wiki/Manufacturing" title="Manufacturing">production</a>, <a href="http://en.wikipedia.org/wiki/Trade" title="Trade">exchange</a>, ], <a href="http://en.wikipedia.org/wiki/Natural_resource" title="Natural resource">natural resource</a> endowment, and <a href="http://en.wikipedia.org/wiki/Ecology" title="Ecology">ecology</a>, as main factors. These factors give context, content, and set the conditions and parameters in which an economy functions.</p>
<p>Today the range of fields of study examining the economy include <a href="http://en.wikipedia.org/wiki/Social_sciences" title="Social sciences">social sciences</a> such as <a href="http://en.wikipedia.org/wiki/Economics" title="Economics">economics</a>, <a href="http://en.wikipedia.org/wiki/Sociology" title="Sociology">sociology</a> (<a href="http://en.wikipedia.org/wiki/Economic_sociology" title="Economic sociology">economic sociology</a>), <a href="http://en.wikipedia.org/wiki/History" title="History">history</a> (<a href="http://en.wikipedia.org/wiki/Economic_history" title="Economic history">economic history</a>), <a href="http://en.wikipedia.org/wiki/Anthropology" title="Anthropology">anthropology</a> (<a href="http://en.wikipedia.org/wiki/Economic_anthropology" title="Economic anthropology">economic anthropology</a>) and <a href="http://en.wikipedia.org/wiki/Geography" title="Geography">geography</a> (<a href="http://en.wikipedia.org/wiki/Economic_geography" title="Economic geography">economic geography</a>). Practical fields directly related to the human activities involving <a href="http://en.wikipedia.org/wiki/Production" title="Production">production</a>, <a href="http://en.wikipedia.org/wiki/Distribution" title="Distribution">distribution</a>, <a href="http://en.wikipedia.org/wiki/Trade" title="Trade">exchange</a>, and <a href="http://en.wikipedia.org/wiki/Consumption" title="Consumption">consumption</a> of <a href="http://en.wikipedia.org/wiki/Goods_and_services" title="Goods and services">goods and services</a> as a whole, range from <a href="http://en.wikipedia.org/wiki/Engineering" title="Engineering">engineering</a> to <a href="http://en.wikipedia.org/wiki/Management" title="Management">management</a> and <a href="http://en.wikipedia.org/wiki/Business_administration" title="Business administration" class="mw-redirect">business administration</a> to <a href="http://en.wikipedia.org/wiki/Applied_science" title="Applied science">applied science</a> to <a href="http://en.wikipedia.org/wiki/Finance" title="Finance">finance</a>.</p>
<p>All kind of <a href="http://en.wikipedia.org/wiki/Profession" title="Profession">professions</a>, occupations, <a href="http://en.wikipedia.org/wiki/Economic_agent" title="Economic agent" class="mw-redirect">economic agents</a> or economic activities, contribute to the economy. <a href="http://en.wikipedia.org/wiki/Consumption_(economics)" title="Consumption (economics)">Consumption</a>, <a href="http://en.wikipedia.org/wiki/Saving_(money)" title="Saving (money)" class="mw-redirect">saving</a> and <a href="http://en.wikipedia.org/wiki/Investment" title="Investment">investment</a> are core variable components in the economy and determine market equilibrium. There are three main sectors of economic activity: <a href="http://en.wikipedia.org/wiki/Primary_sector_of_the_economy" title="Primary sector of the economy">primary</a>, <a href="http://en.wikipedia.org/wiki/Secondary_sector_of_the_economy" title="Secondary sector of the economy">secondary</a> and <a href="http://en.wikipedia.org/wiki/Tertiary_sector_of_the_economy" title="Tertiary sector of the economy">tertiary</a>.</p>
