<p><b>Micro</b> is an English prefix of Greek origin that is most commonly known for its usage in the word <a href="http://en.wikipedia.org/wiki/Microscopic" title="Microscopic" class="mw-redirect">microscopic</a>, the prefix refers to an object as being smaller than scale of focus, in contrast with <a href="http://en.wikipedia.org/wiki/Macroscopic" title="Macroscopic" class="mw-redirect">macro</a>. Some examples of the prefix "micro" are <a href="http://en.wikipedia.org/wiki/Microwave" title="Microwave">microwave</a>, <a href="http://en.wikipedia.org/wiki/Microphone" title="Microphone">microphone</a>, <a href="http://en.wikipedia.org/wiki/Integrated_circuit" title="Integrated circuit">microchip</a> and <a href="http://en.wikipedia.org/wiki/Microorganism" title="Microorganism">micro-organism</a>.</p>
<ul>
<li><b><a href="http://en.wikipedia.org/wiki/Micro-" title="Micro-">micro-</a></b> is a prefix in the <a href="http://en.wikipedia.org/wiki/SI" title="SI" class="mw-redirect">SI</a> and other systems of units denoting a factor of 10<sup>−6</sup> (one millionth). Symbol, µ
<ul>
<li>Micro as shorthand for an SI based measurement, in the case of <a href="http://en.wikipedia.org/wiki/Blood_lead_level" title="Blood lead level">blood lead level</a> for example, 10 micros would be 10 micrograms per deciliter.</li>
</ul>
</li>
<li>A shortened reference to <a href="http://en.wikipedia.org/wiki/Microeconomics" title="Microeconomics">microeconomics</a>, <a href="http://en.wikipedia.org/wiki/Microsociology" title="Microsociology">microsociology</a>, <a href="http://en.wikipedia.org/wiki/Microbiology" title="Microbiology">microbiology</a> or similar fields of science</li>
<li><a href="http://en.wikipedia.org/wiki/Microcomputer" title="Microcomputer">Microcomputer</a> (semi-obsolete usage)</li>
<li><a href="http://en.wikipedia.org/wiki/Microcontroller" title="Microcontroller">Microcontroller</a>, electronics, microcontroller (also MCU or µC) is a functional computer system-on-a-chip</li>
<li><a href="http://en.wikipedia.org/wiki/Micromanagement_(computer_gaming)" title="Micromanagement (computer gaming)" class="mw-redirect">Micromanagement (computer gaming)</a></li>
<li><a href="http://en.wikipedia.org/wiki/Micro_DBMS" title="Micro DBMS">Micro DBMS</a>, an early relational database management system</li>
<li>The <a href="http://en.wikipedia.org/wiki/Micro_programming_language" title="Micro programming language">Micro programming language</a>, a simple programming language used in teaching <a href="http://en.wikipedia.org/wiki/Compiler_construction" title="Compiler construction">compiler construction</a> techniques</li>
<li><a href="http://en.wikipedia.org/wiki/Microchip_(comics)" title="Microchip (comics)">Microchip (comics)</a>, a supporting character in <i><a href="http://en.wikipedia.org/wiki/The_Punisher" title="The Punisher" class="mw-redirect">The Punisher</a></i> comics (often known as Micro)</li>
<li><a href="http://en.wikipedia.org/wiki/Mikro" title="Mikro">Mikro</a>, a <a href="http://en.wikipedia.org/wiki/Greece" title="Greece">Greek</a> <a href="http://en.wikipedia.org/wiki/Electronic_music" title="Electronic music">electronic music</a> group</li>
<li><a href="http://en.wikipedia.org/wiki/Micro_(car)" title="Micro (car)">Micro (car)</a> is the first car company from Sri Lanka</li>
<li>"The Micro" is an Argentinean and Chilean slang term for buses in the public transportation system (as in "I'm waiting for the micro,").</li>
<li><a href="http://en.wikipedia.org/wiki/Microhouse" title="Microhouse">Microhouse</a>, a subgenre of <a href="http://en.wikipedia.org/wiki/House_music" title="House music">House music</a></li>
</ul>
